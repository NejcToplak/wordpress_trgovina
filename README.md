# wordpress_trgovina
4. predmet poklicne mature - wordpress trgovina
Mini spletna trgovina

4. predmet poklicne mature | WordPress + WooCommerce | 4. letnik SERŠ

## Opis projekta
Funkcionalna mini spletna trgovina z računalniško opremo,
izdelana z WordPress in WooCommerce vtičnikom.

## Tehnologije
- WordPress 6.x
- WooCommerce
- PHP 8.x
- CSS3
- Storefront Child tema

## Kaj vsebuje projekt
- Prilagojena child tema (style.css, functions.php)
- Katalog izdelkov s kategorijami in variacijami
- Nakupovalna košarica in blagajna
- Registracija in prijava uporabnikov
- Kontaktni obrazec (Contact Form 7)
- SEO optimizacija (Yoast SEO)
- Varnostni ukrepi (Wordfence)

## Namestitev lokalno
1. Namesti XAMPP
2. Namesti WordPress v `htdocs/trgovina/`
3. Namesti WooCommerce vtičnik
4. Kopiraj mapo `storefront-child` v `wp-content/themes/`
5. Aktiviraj temo v WordPress adminu
